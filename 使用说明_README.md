# 中国陶瓷文物数字展厅 · 完整交付包

本文件夹包含：**源代码**、**已构建页面（`dist`）**、**63 件器物高清图**（`public/images/relics`）及依赖清单。  
发给他人时请**整包压缩发送**，解压后按下列任一方式打开即可看到相同效果。

---

## 环境要求

- **Node.js 18 或以上**（长期维护版即可）  
  下载：https://nodejs.org/zh-cn/  
  安装后在命令行执行 `node -v` 能显示版本号即表示成功。

> 说明：本站点为现代前端工程，需要通过本地小型 HTTP 服务访问（不能直接双击 `index.html` 打开，否则图片与脚本路径会异常）。

---

## 方式一：最快预览（推荐给只浏览效果的老师/同学）

适合：**不想改代码**，只想和作者电脑上看到的一样。

1. 解压本文件夹到任意英文或中文路径均可。  
2. **双击运行 `一键打开网站.bat`**。  
3. 首次运行会自动下载轻量静态服务工具（需联网一次），完成后浏览器访问：  
   **http://localhost:4173**  
4. 关闭黑色命令行窗口即停止网站。

若 4173 端口被占用，可编辑 `一键打开网站.bat` 最后一行数字改为其他端口（如 4174）。

---

## 方式二：开发模式（适合答辩现场改数据或演示热更新）

1. 在本文件夹内打开终端（PowerShell 或 CMD）。  
2. 执行：

```bash
npm install
npm run dev
```

3. 按终端提示在浏览器打开 **http://localhost:5173**（可能为其它端口，以终端为准）。

---

## 方式三：重新打包发布

```bash
npm install
npm run build
npm run serve-dist
```

构建结果在 `dist` 目录；`serve-dist` 会在 **http://localhost:4173** 提供预览。

## 文件夹说明（提交检查清单）

| 内容 | 说明 |
|------|------|
| `src/` | React 源码（`App.jsx`、数据与分面逻辑） |
| `依据/` | **唯一资料源**：汇报表（器物与编码）+《陶瓷器物图片分类卡片版》（图像与原图序号） |
| `public/images/relics/` | 63 张图由 `npm run images:from-yiju` 从《依据》内分类卡片文档提取 |
| `src/data/rawObjects.json` | 由 `npm run data:from-yiju` 从《依据》内汇报表导出 |
| `dist/` | 已构建的静态网站，供方式一直接预览 |
| `package.json` / `package-lock.json` | 依赖锁定，保证他人 `npm install` 版本一致 |
| `vite.config.js` | 已设置 `base: './'`，便于任意目录部署 |

## 常见问题

**Q：双击 bat 一闪而过？**  
A：未安装 Node.js 或未加入 PATH。请先安装 Node 并重启电脑。

**Q：图片不显示？**  
A：请确认使用方式一或方式二、三启动，**不要**用资源管理器直接打开 `dist/index.html`。

**Q：字体与作者略有不同？**  
A：需联网加载 Google Fonts；离线时自动回退为系统宋体/黑体，不影响功能。
